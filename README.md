## Overview
The **RandomlyRight** is a web-based application that provides users with insightful advice at the click of a button. Users can react to advice, report inappropriate content, and share advice on social media. The platform ensures engagement through a sleek, modern UI with a dark theme.

ðŸ”— **Live Project URL:** [RandomlyRight](https://randomlyright.vercel.app)

## Features
- **Authentication:** Users must log in to access advice.
- **Random Advice Display:** Advice fetched dynamically from Firebase.
- **Reaction Panel:** Users can react using emojis.
- **Report System:** Users can flag inappropriate advice.
- **Auto-Refresh Toggle:** Fetches new advice every 10 seconds.
- **Social Sharing:** Users can share advice on different platforms.
- **User Profile:** Displays user details and reacted advice.
- **Add Advice:** Users can submit advice with categories.

## Tech Stack
- **Frontend:** React.js, Bootstrap, CSS Animations
- **Backend:** Firebase Firestore (Database), Firebase Authentication
- **Hosting:** Vercel

## Usage
1. **Sign Up/Login:** Users need to log in via email/password or Google authentication.
2. **View Advice:** Random advice is displayed on the main page.
3. **React & Report:** Users can react or report inappropriate advice.
4. **Add Advice:** Users can contribute their own advice by entering text and selecting a category.
5. **Profile Page:** Displays the user's details and the advice they've reacted to.
